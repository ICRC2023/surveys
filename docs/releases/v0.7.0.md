# v0.7.0 - Pluggable Survey Framework & Plugin System (2026-04-09)

## What Changed?

This release transforms Titanite from an ICRC2023-specific tool into a **pluggable, generic Google Forms survey processing framework**. The new plugin architecture enables support for multiple surveys without modifying core code. All existing ICRC2023 workflows remain fully compatible.

---

## What's New

### Main Feature: Plugin Architecture

**What it does**:
- Abstract `SurveySchema` base class for defining survey-specific rules
- `SurveyProcessor` pipeline with 8-step processing
- Dynamic plugin loading via `--plugin` CLI option
- `ICRC2023Schema` reference implementation

**How to use it**:

```bash
# Default ICRC2023 workflow (backward compatible)
cd sandbox
poetry run ti prepare ../data/raw_data/survey.csv

# With explicit plugin
poetry run ti prepare ../data/raw_data/survey.csv --plugin plugins.icrc2023.ICRC2023Schema

# Custom plugin
poetry run ti prepare ../data/raw_data/custom_survey.csv --plugin plugins.custom.CustomSchema
```

**To create a new survey plugin**:
See [PLUGIN_DEVELOPMENT_GUIDE.md](../../PLUGIN_DEVELOPMENT_GUIDE.md) for detailed instructions.
